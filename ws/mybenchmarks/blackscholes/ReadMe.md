### Black scholes benchmark

Usage:

	script(n)
	% n can be
	% 0: original
	% 1: vector
	% 2: parfor
	% 3: optimized by hand

Files:

- script
  - script_*.m
    - run\_script_*.m
      - runBlkSchls_*.m


- runBlkSchls_new.m: source code
- runBlkSchls_vector.m: vectorized code
- runBlkSchls_parfor.m: parfor code
- runBlkSchls_opt.m: hand optimized code
